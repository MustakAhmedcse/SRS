## 3. KPI 1 — GA Variable Incentive (Core KPI)
The primary KPI across all 14 standard BP Variable SRFs. A BP earns variable incentive BDT per additional GA beyond their monthly target, up to a capped maximum.
[TABLE]
| KPI Name | GA Variable Incentive |
| Trigger | After achieving monthly GA target, each additional GA earns a fixed BDT rate |
| Assessment Window | Only first N GA of the month are assessed (both fixed + variable salary) |
| Variable Range | Post-target GA range defines the eligible additional GA (e.g., 51–120 GA) |
| Max Additional GA | Capped at a maximum number of extra GA for variable pay |
| Rate Unit | BDT per additional GA (varies by BP type, market type, and month) |
[/TABLE]

## 3.3 Cluster-Based GA Attribution: FCD vs FCD+SMART POS
GA attribution method varies by cluster. Two methods were used, and which clusters used which method evolved over the months:
[TABLE]

| Method B: FCD + SMART POS (CODE: ARRANGED-3) | GA = Designated Retailer FCD data OR SMART POS data, whichever is higher. Used where SMART POS coverage supplements FCD data. |
| SMART POS Code | ARRANGED-3 |
| BP Retailer Code Source | B2C shares BP-wise retailer codes |
[/TABLE]
[TABLE]
| Month | FCD Only Clusters | FCD + SMART POS Clusters |

| May–Jun 2025 | All clusters (BP Wise Retailer code from B2C; no cluster split) | — |

| Aug 2025 (original) | North, North East, West Cluster | East, Central Cluster |

## 3.5 VLR Condition — Three-Tier Payout Slab
From October 2025, a VLR (Visitor Location Register) active condition was added as a quality gate on GA counting. The VLR condition uses a 3-tier payout slab:
[TABLE]
| VLR Condition Introduced | October 2025 (partial month: 22nd–31st Oct only) |
| VLR Condition Full Month | November 2025 onward — applies to entire campaign period |
| VLR Data Source | B2C shares VLR achievement % per BP |
[/TABLE]
[TABLE]
| VLR Act % vs All GA | Standard SRFs (Oct–Jan 2026) | Feb 2026 onward (raised threshold) |
| Tier 1 (Full) | ≥80% → Full GA Achievement | ≥85% → Full GA Achievement |
| Tier 2 (Partial) | 60%–<80% → VLR Active GA Only | 65%–<85% → VLR Active GA Only |
| Tier 3 (None) | <60% → No GA will be Considered | <65% → No GA will be Considered |
[/TABLE]
Note: 'VLR Active GA Only' means only those specific activations where the subscriber is on VLR Active status are counted — not the full GA total. This is more selective than the full count.

Note: The Central Cluster distinction (BDT 120 vs BDT 100 for other clusters) reflects the higher competitive intensity and activation volumes expected in urban Central Cluster territories.

## 5. KPI 3 — GA Quality (48hr Usage) Incentive
A 48-hour usage quality incentive was added from May 2025:
[TABLE]
| KPI Name | GA Quality — 48 Hour Usage Incentive |
| Applicable Period | May 2025 – October 2025 (not present in Apr 2025; dropped Nov 2025 onward when VLR replaced it) |
| 48hr Usage Definition | Any revenue-generating activity within 48 hours of SIM activation |
| Minimum Threshold | Minimum 75% of total activations must be 48hr-usage-qualified |
| Payout Basis | Prorated based on achievement % (below 75% = zero; above = proportional) |
| Maximum Incentive | BDT 1,000 |
| Data Source | Retailer-wise total activation and qualified 48hr count shared by B2C |
[/TABLE]
Note: The GA Quality incentive (48hr usage, max BDT 1,000) was effectively replaced by the VLR condition from Oct 2025. The VLR condition addresses the same quality objective (active usage post-activation) but uses VLR active status instead of 48hr revenue activity.

## 7. KPI 5 — FR (First Recharge) Incentive (Apr 2025 only)
April 2025 had a unique FR-linked variable incentive structure distinct from all other months:
[TABLE]
| KPI Name | FR-Qualified GA Variable Incentive |
| Applicable Period | April 2025 only |
| FR Codes | 48, 81, 222, 224, 104 (specific First Recharge denomination codes) |
| Condition | Each additional GA (11th–20th) must have FR from one of the specified codes (48/81/222/224/104) AND be within 1–30 Apr 2025 |
| Rate | BDT 250 per FR-qualified GA |
| Variable Range | 11–20 GA (after hitting monthly GA target) |
| Max Additional GA | 10 (only 10 additional FR-qualified GA counted) |
| Max Assessed GA | First 20 GA only assessed for both fixed and variable |
| GA Attribution | Higher of: BP Wise Retailer FCD OR BP Wise SMART POS (ARRANGED-3) |
| Usage Quality | Customer must use ≥10 minutes or 500 MB within assessment window |
[/TABLE]
Note: April 2025 is the only month where the variable incentive was tied to specific First Recharge denominations rather than any GA. The low cap (10 extra GA) and high rate (BDT 250/GA) make this the most targeted and highest per-GA rate in the entire BP Variable series.

## 9. Sub-Campaign Type 2 — Freelancer BP Variable Incentive (Central Cluster)
3 SRFs covering Freelancer BPs in the Central Cluster only (Feb–Apr 2026). Freelancer BPs have a fundamentally different incentive structure from standard BPs.
## 9.1 Freelancer BP Structure — Key Differences
[TABLE]
| Attribute | Standard BP Variable | Freelancer BP Variable (Central Cluster) |
| Target Structure | Monthly GA target provided by B2C; variable pay after target | No monthly target — minimum absolute GA threshold instead |
| Minimum GA | Varies (dependent on target) | 50 GA minimum to qualify for any variable pay |
| Maximum GA Counted | 120–165 GA (varies by month) | 150 GA (fixed) |
| Rate | BDT 85–120/GA (varies) | BDT 100/GA for ALL GA if total ≥50 |
| GA Range | Post-target (e.g., 71–165 GA) | All GA from 1st GA if total ≥50 (no post-target threshold) |
| BP Types | Regular/LUS or Core/Emerging or New/Existing | No sub-type classification |
| Cluster | All clusters | Central Cluster only |
| Daily Productivity | Present (Feb–Apr 2026) | Not included |
| MyBL/Toffee | Included | Not included |
| GA Quality (48hr) | Included (May–Oct 2025) | Not included |
[/TABLE]
Note: For Apr 2026, the Freelancer BP SRF clarified: 'Incentive will be BDT 100 PER GA for all GA if total GA ≥50. Example: If total GA = 55, then commission = 55 × BDT 100 = BDT 5,500 (VLR condition applies)'. This confirms the rate applies to ALL GA (not just post-50 GA).
