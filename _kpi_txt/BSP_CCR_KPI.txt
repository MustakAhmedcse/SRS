[TABLE]
| Recipient | BSP CCR (Field Force CCR) |
| Base Variable Pay | BDT 4,000 / month (performance-based) |
| Effective Period | Apr 2025 → Jan 2026 onwards (ongoing with revisions) |
| KPI Structure | Weighted KPI gates — must meet threshold per gate to earn that portion |

Key rule: Each KPI gate has an independent threshold. If threshold is NOT met for a gate, that gate's weightage portion pays zero. Remaining gates pay at their respective achievement rates.

5.1 Payout Slabs by SRF Version
[TABLE]
| SRF Version | Threshold | Payout Structure |
| Apr 2025 | 90% | >=90% ach → full incentive. No sub-slabs mentioned. 10% kept as reserve for system issues. |
| May & Jul 25 | 80% | >=80% – <100% = At Actual Payout (pro-rated); >=100% = 100% Payout |
| Oct 2025 | 80% | Target Ach 15%: <80%=0%; 80-84.99%=60%; 85-89.99%=70%; 90-94.99%=80%; 95-<100%=90%; >=100%=100%. First Recharge 15%: 60-85% ach = pro-rated; >=85% = 100%. Rolling 45 days of acquisition. |
| Jan 2026 | 93% | Target Ach 15%: <93%=0%; >=93% <101%=95%; >=101% <108%=110%; >=108%=120%. First Recharge 10%: 60-85% = pro-rated; >=85%=100%. Rolling 45 days. |
[/TABLE]
Note: Oct 2025: GA KPI is split into two sub-components — Target Achievement (15%) and First Recharge (15%) — totalling 30% weight for GA gate.
Note: Jan 2026: GA KPI sub-components become Target Achievement (15%) and First Recharge (10%) — total 25% weight.
6. First Visit Resolution (FVR) of Core Services — KPI
[TABLE]
| Definition | Agent resolves customer issue on first visit (core service categories) |
| Measurement | Unique work code vs. total work code for that agent on same day |
| Threshold | 90% (Apr–Jul 2025) | 90% (Oct 2025) | 95% (Jan 2026+) |
| Weight | 20% (Apr–Jul 2025) | 10% (Oct 2025 onwards) |
| Uniqueness Rule | Customer MSISDN considered unique for the same service on the same day |
| Pass condition | Unique work code count >= threshold % of total actualization work code |
| Fail condition | Unique work code < threshold % → 0% payout for this gate |
| System Reserve | Apr 2025: 10% of total incentive kept as reserve for system-related issues (exception notified by email) |
| Target Source | Shared by Business/User team |
[/TABLE]

12.2 BP Replacement & Eligibility Policy
Applicable only for 18 specified centers where dedicated TechVan BP is NOT present.
NO new TechVan BP will be appointed for any new center (except Experience Centers such as Tiger's Den with dedicated device display table).
If existing TechVan BP resigns: position is NOT replaced (except Experience Centers). That center's Agent/CCR becomes eligible from the resign date.
Experience Centers MUST continue to have TechVan BP — CCRs at Experience Centers are NOT eligible for this incentive.
Monthly eligible CCR list shared by HQ Team.
