3. Commission Model Types — Classification
[TABLE]
| Model Type | Description | Trigger | Cap | Partners |
| Regular Deno-Based | Fixed BDT commission per transaction on specified deno(s). Different denos may apply in different sub-periods within the campaign. | Any qualifying recharge of that deno from authorized MSISDN | No ceiling | bKash, Nagad, Rocket, Upay |
| Hourly / Time-Gated | Commission applies ONLY during a specified time window (e.g., 4PM–6PM) on qualifying denos. | Recharge must occur within the hourly window | No ceiling (unless stated) | bKash, Nagad, Rocket, Upay |
| Flash Drive (Time+Date) | Like hourly but applies to specific dates AND hours. Date-time combination gates eligibility. | Date AND time must both match | No ceiling | bKash |
| Winback / Segmented Base | Commission only for recharges to a pre-selected subscriber base. Per-subscriber, per-campaign limit (usually once). IRIS excluded. | Specific deno + selected base subscriber | Once per subscriber per campaign | bKash, Nagad |
| 1st Time Recharge | Commission for recharges to subscribers who are receiving the deno for the first (or Nth) time. Subscriber-level counter applied. | Subscriber's Nth recharge of that deno in the period | Defined count per subscriber (1x or 2x) | bKash, Nagad |
| Self-Recharge Campaign | Subscriber recharges their own number (not retailer-to-subscriber). Same deno-based commission structure. | Self-recharge of specified deno | No ceiling | bKash |
| Sequential / Tiered | Two-tier commission: 1st recharge gets amount A, 2nd recharge gets amount B for same subscriber and deno. | Subscriber's 1st then 2nd recharge of deno | 2 times per subscriber | bKash |
| Minute-Winner / CB Drive | Only the fastest recharger(s) per minute or per hour wins. Winner count limited per hour/day. | 1st recharger of the minute or hour with specified deno | Max winners per hour/day defined | bKash |
| 100% Cashback (CB) Drive | Subscriber effectively receives cashback equal to recharge amount. Commission = deno amount. | 1st recharger per minute wins; 120 winners/day | 1 winner per minute, 120/day | bKash |
| Volume-Based % Commission | Commission = fixed % of total recharge volume during campaign. Requires hitting a recharge target. | Total recharge during period from authorized MSISDN | Max BDT 40,000 ceiling applies | Techno Index |
[/TABLE]
4. bKash Commission Conditions
4.1 Regular Bi-Monthly Drive (Standard Deno-Based)
The most common drive type. bKash receives a fixed BDT commission per qualifying recharge on specified denos. Campaign is split into two sub-periods (e.g., 1st–7th and 8th–15th) with different deno-commission tables.
[TABLE]
| Condition | Details |
| Recipient | Digital Distributor — bKash |
| Commission Basis | Fixed BDT amount per transaction on specified deno(s) |
| Sub-Period Split | Typically 2 sub-periods within each half-month (e.g., 1–7 and 8–15). Each sub-period has its own deno list and commission rates. |
| Ceiling | No maximum or minimum ceiling |
| IRIS Transactions | Excluded — no commission on IRIS transactions |
| I-top Definition | I-top sales = sales to end customer only; not lifting |
| Recharge Source | Must originate from authorized bKash retailer EV MSISDNs (21 MSISDNs listed) |
| Transaction Type | Recharge (retailer EV to subscriber) — EV/I-TOP method |
| Maturity | Campaign end date + 1 day (e.g., period ends 15-May, maturity 16-May) |
| Data Source | IT SC + EV Platform + DMS + POS |
| Report Format | Customer_MSISDN | RET_MSISDN | RECHARGE_AMOUNT | TOP_UP_DATE | HIT | TOTAL_RECHARGE | COMMISSION_AMOUNT |
[/TABLE]
Note: Deno lists and commission amounts change every period. The commission structure (fixed per deno) is consistent.
4.2 Special Drive
Same structure as Regular Drive but covers specific high-value or promotional denos (often higher-denomination bundles like 497, 729, 998, 1258). Same conditions apply.
[TABLE]
| Condition | Details |
| Recipient | Digital Distributor — bKash |
| Commission Basis | Fixed BDT per transaction on special promotional denos |
| Typical Denos | High-value: 497, 729, 998, 1258 and similar bundle denos |
| Sub-Period Split | Yes — typically 4 sub-periods (1–3, 6–7, 9–11, 13–14 May pattern) |
| Ceiling | No maximum or minimum ceiling |
| IRIS Transactions | Excluded |
| Recharge Source | Authorized bKash retailer EV MSISDNs only |
| Maturity | Campaign end date (same-day maturity in some cases) |
| All other conditions | Identical to Regular Drive |
[/TABLE]
4.3 USSD Drive
Targets a specific deno associated with USSD recharge behaviour (e.g., deno 335). Single deno, simple structure.
[TABLE]
| Condition | Details |
| Commission Basis | Single fixed deno; one commission rate for the entire period |
| Typical Deno | 335, 275, 325 (varies by period) |
| Sub-Period Split | Sometimes split (e.g., Aug: 8–14 @ 335, 15–23 @ 275, 24–31 @ 325) |
| Ceiling | No maximum or minimum ceiling |
| IRIS Transactions | Excluded |
| All other conditions | Identical to Regular Drive |
[/TABLE]
4.4 Hourly / Flash Drive (Time-Gated)
Commission applies ONLY within a defined time window (typically 4:00 PM – 5:59 PM or 4:00 PM – 6:00 PM). Outside those hours, no commission is earned even on qualifying denos.
[TABLE]
| Condition | Details |
| Time Window | Typically 4:00:00 PM to 05:59:59 PM (2-hour window). Some variants: 5:00 PM–7:00 PM, 3:00 PM–6:00 PM. |
| Granularity | Seconds-level: 16:00:00 to 17:59:59 (inclusive) |
| Commission Basis | Fixed BDT per qualifying recharge within the time window |
| Date Scope | Can span multiple days (e.g., 27–30 Apr) or specific individual dates |
| Typical Denos | Low-to-mid: 79, 124, 147, 169, 174, 199, 998 and similar |
| Ceiling | No maximum or minimum ceiling |
| IRIS Transactions | Excluded |
| Recharge Source | Authorized bKash retailer EV MSISDNs only |
| Report includes | Both Gross Commission (including AIT) and Net Commission (without AIT) reported |
| All other conditions | Identical to Regular Drive |
[/TABLE]
Note: Flash drives are essentially hourly drives with specific date+time combinations. The commission table may include both a Gross Commission column and a Gross Commission (including AIT) column — verify which column to use for payout.
4.5 Winback Campaign (Segmented Base)
Commission is limited to recharges made to a pre-selected ('selected base') list of subscribers. Retailer earns commission only once per subscriber per campaign period.
[TABLE]
| Condition | Details |
| Recipient | Digital Distributor — bKash |
| Base Restriction | Selected/segmented subscriber base only. Non-base subscribers do NOT qualify. |
| Per-Subscriber Limit | Commission payable ONCE per subscriber per campaign period |
| Consumer Incentive Rule | Each consumer gets incentive ONE TIME ONLY for the specified deno recharge |
| Typical Deno | Small denos: 20 (most common) |
| Campaign Length | ~7–14 days (e.g., 25–31 Oct, 18–31 Jan, 8–31 May, 24–30 Jun) |
| IRIS Transactions | Excluded |
| Ceiling | No ceiling on total payouts; ceiling is per-subscriber limit |
| Standard General Conditions | Recharge-based (EV/I-TOP). Same maturity rules as regular drive. |
| Mar 2026 Version | Uses FCD activation gate + ETSAF SAF condition + Prepaid product scope (see Section 10 for full comparison) |
[/TABLE]
Note: Winback Nov 2025 onwards uses a newer SRF format (requester: Sarzanul Haque Sabid) with additional activation gate (FCD) and enhanced SAF conditions (98% ETSAF). Earlier Winback SRFs use the standard format.
4.6 1st Time Recharge Campaign
Commission triggered by a subscriber's 1st (or 2nd) recharge of a specific deno during the campaign. A sequential commission tier may apply.
[TABLE]
| Condition | Details |
| Typical Structure | Two-tier: 1st recharge of deno = commission rate A; 2nd recharge of same deno = commission rate B |
| Per-Subscriber Limit | Defined count (e.g., 2 times per subscriber). After 2nd recharge, no more commission for that subscriber. |
| Example (Jul 2025) | Deno 29: 1st recharge = BDT 10 net (11.11 gross), 2nd recharge = BDT 15 net (16.67 gross) |
| Base Restriction | Selected base considered (segmented base) |
| Typical Deno | Small denos: 29, 35 |
| IRIS Transactions | Excluded |
| Ceiling | No ceiling — capped by per-subscriber limit only |
| All other conditions | Same as Winback / Segmented campaign |
[/TABLE]

[TABLE]

4.8 Sequential / Tiered Recharge Campaign
Similar to 1st Time Recharge but explicitly labelled 'sequential'. Subscriber gets a lower commission for their 1st qualifying recharge and a higher amount for their 2nd.
[TABLE]
| Condition | Details |
| Tier Structure | 2 tiers: 1st recharge = lower rate; 2nd recharge = higher rate |
| Per-Subscriber Limit | Exactly 2 incentives per subscriber; no further commission after 2nd |
| Example (Aug 2025) | Deno 35: 1st = BDT 9.6 net (10.67 gross), 2nd = BDT 12 net (13.34 gross) |
| Base Restriction | Selected/segmented base |
| IRIS Transactions | Excluded |
| Ceiling | Capped by per-subscriber count only |
[/TABLE]
4.9 Minute-Winner / CB Drive (Fastest Recharger)
Only the FASTEST recharger(s) in each minute (or hour) wins the commission. A cap on total winners per day/hour applies.
[TABLE]
| Condition | Details |
| Winner Selection | 1st recharger per minute = winner. OR: 1st N rechargers per hour = winners. |
| Daily Winner Cap | Defined per SRF (e.g., 180 winners/day, max 60 per hour) |
| Hourly Winner Cap | Max winners per hour specified (e.g., first 20 customers per hour in Sep 2025 morning drive) |
| Time Windows | Multiple time windows may apply in same SRF (e.g., 10AM–10PM hourly winners + 3PM–6PM minute winners) |
| Typical Denos | 156 (Sep 2025), others vary |
| Ceiling | No monetary ceiling but winner count cap acts as the ceiling |
| IRIS Transactions | Excluded |
| All other conditions | Same as regular drive for non-winner conditions |
[/TABLE]
4.10 100% Cashback (CB) Drive
Subscriber receives a commission equal to the recharge deno amount. Fastest recharger per minute wins. 120 winners per day maximum.
[TABLE]
| Condition | Details |
| Commission = Deno Amount | Commission per winner = deno value (100% cashback to customer via retailer) |
| Winner Rule | Only 1st recharger per minute eligible. One winner per minute. |
| Daily Cap | 120 winners per day (= 120 minutes of 2-hour window) |
| Time Window | 4:00:00 PM to 5:59:59 PM (2 hours = 120 minutes) |
| Example (May 2025) | Deno 29: Gross commission 32.22 (including AIT), Net 29 |
| IRIS Transactions | Excluded |
| All other conditions | Same as regular drive |
[/TABLE]
4.11 Asia Cup / Event Flash Drive
Same as Hourly/Flash Drive but tied to a sporting event (Asia Cup). Specific match-day dates and time windows.
[TABLE]
| Condition | Details |
| Time Windows | Match-specific hours (e.g., 5PM–7PM, 4PM–6PM on specific match dates) |
| Dates | Individual match dates (not continuous period) |
| Example (Sep 2025) | Deno 169, BDT 5 net (5.56 gross) — specific days: 11th, 13th, 14th, 16th Sep |
| All other conditions | Identical to standard Flash/Hourly Drive |
[/TABLE]

11. Input Data Sources & Report Format

| DMS (Distribution Management System) | All SRFs | Yes |

[/TABLE]
11.2 Standard Report Output Format
[TABLE]
| Field | Description |
| Customer_MSISDN | Subscriber's mobile number (the recharge recipient) |
| RET_MSISDN | Retailer's EV MSISDN (must be in the authorized list) |
| RECHARGE_AMOUNT | Amount recharged (must match qualifying deno) |
| TOP_UP_DATE | Date of the recharge transaction |
| HIT | Whether the transaction qualifies (hit/miss indicator) |
| TOTAL_RECHARGE | Total recharge amount for the subscriber in the period |
| COMMISSION_AMOUNT | Commission earned for that transaction |
[/TABLE]
Note: Techno Index report excludes Customer_MSISDN. Hourly drives may have additional Time column for time-window validation.
12. Key Definitions & Glossary
[TABLE]
| Term | Definition |
| Digital Distributor | A partner organization (bKash, Nagad, etc.) acting as a digital recharge distribution channel. Commission recipient in all Digital SRFs. |
| Deno | Denomination — the specific recharge amount (e.g., BDT 29, 199, 729). Commission rates are deno-specific. |
| EV / I-TOP | Electronic Voucher / I-Top — the valid recharge methods. Only EV and I-TOP transactions count; IRIS and lifting are excluded. |
| IRIS Transaction | An internal transaction type excluded from bKash commission calculations. Not always stated for other partners. |
| Lifting | Distributor-to-retailer stock transfer — NOT considered a sale to end customer and NOT eligible for commission. |
| Authorized MSISDN | The specific retailer EV MSISDNs through which recharges must be made to qualify. bKash has 21, Nagad 3, Rocket 3, Upay 1, Techno Index 1. |
| Gross Commission (incl. AIT) | Commission amount inclusive of Advance Income Tax (AIT). This is the amount Banglalink pays. |
| Net Commission (excl. AIT) | Commission amount excluding AIT — the amount the partner actually retains after tax. |
| AIT | Advance Income Tax — deducted from commission at source. Gross = Net + AIT. |
| Segmented Base | A pre-selected list of subscriber MSISDNs targeted in Winback/1st Recharge campaigns. Only recharges to these subscribers qualify. |
| FCD | First Call Date — the date on which a subscriber first makes a call. Used as an activation gate in newer Winback SRFs. |
| FAD | First Activity Date — alternative activation gate (not used in any checked Digital SRF). |
| ETSAF | End-to-end SAF compliance metric. In Winback SRFs: 98% of activations must be vetted within 7th of following month. |
| SAF | Subscriber Acquisition Form — subscriber verification process. DMS Post Date within 7 days of activation; Vetting by 14th of following month (standard). |
| R2RT | Retailer-to-Retailer Transfer — one form of EV recharge in expanded Winback definition. |
| D2RT | Distributor-to-Retailer Transfer — another form of EV recharge in expanded Winback definition. |
| DTR | Daily Transaction Report — manual and daily commission component from SalesCom, included in Winback recharge definition. |
| OTF | Over The Form — EV OTF and IRIS OTF, included in Winback recharge definition. |
| Minute-Winner | The subscriber who recharges first within a given minute in a CB drive. One winner per minute. |
| CB Drive | Cashback Drive — subscriber receives cashback (commission equivalent to deno value). 100% CB = commission equals recharge amount. |
| RSO | Retail Sales Officer — recipient description in newer Winback format SRFs. |
| T-6 | Report generation date minus 6 days — used as the cutoff for disconnection data in FCD-format SRFs. |
| Recharge Target | Minimum total recharge volume required to earn commission (used only in Techno Index SRF). |
| Ceiling | Maximum commission cap. No ceiling in most SRFs; BDT 40,000 ceiling in Techno Index only. |
[/TABLE]
