3.2 KPI Framework & Measurement Basis
Campaign Hygiene SRFs do not list explicit KPI weights in the document — the detailed campaign modality and KPI weightage are shared externally by B2C in a separate file. The following metrics are referenced as KPI components:
[TABLE]
| KPI / Metric | How Shared | Notes |
| GA (Gross Add) | Target shared by B2C in Excel | Primary activation KPI. FCD-gated. |
| On Site Strike Rate (SCR) | DH SCR Achievement file shared separately | Distribution House-level strike rate. Gate for eligibility. |
| Active LSO Count | Eligibility file shared by B2C | Active Local Sales Officer count at DD level. |
| Active BSO Count | Eligibility file shared by B2C | Active Business Support Officer count at DD level. |
| SSO / DSO Count | Referenced in May–Aug SRFs (not Apr) | Additional field force role counts added from May 2025. |
| RSO Wise Daily Minimum Transaction Count | Shared in separate mail | RSO productivity threshold. Minimum daily transaction gate. |
| FF GA Participation | Shared by B2C (Aug 2025 only) | Field Force GA Participation metric — added in Aug 2025. |
| Month-end RDB | Auto (system) | Month-end Retail Database — the agent list snapshot used for attribution. |
[/TABLE]
Note: KPI weightage may vary from DD to DD — explicitly stated in all Campaign Hygiene SRFs. Weights are not fixed in the SRF itself.

4.2 KPI: GA Target Achievement
The sole KPI is GA (Gross Add). There is a single achievement threshold required to earn any incentive.
[TABLE]
| Condition | Details |
| KPI | GA (Gross Add) |
| Achievement Threshold | >=100% of Monthly GA Target (standard condition for all months) |
| East Cluster Exception (Jan 2026 Phase 1) | East Cluster: >=100% Ach OR >=70% Ach of Target eligible for slab-wise incentive voucher. Rest Cluster: >=100% only. |
| Report Extraction | DD-wise (per Distribution House) |
| Target Source | Distributor-wise target shared by B2C |
| Agent List Date | Snapshot taken on the campaign end date (e.g., 31st Oct, 31st Dec, 7th Jan, 15th Jan, 25th Jan) |

[/TABLE]
4.3 Quality Gates (48hr Usage & VLR Active)
Two mandatory quality gates must be met before GA activations count toward the target. Both thresholds are provided by B2C.
[TABLE]
| Gate | Threshold (Oct 2025) | Threshold (Nov 2025) | Threshold (Dec 2025) | Threshold (Jan 2026: 1–7, 8–15) | Threshold (Jan 2026: 16–25) |
| 48 Hour Usage | >=85% | >=80% | >=85% | >=85% | >=80% |
| VLR Active Usages | >=85% | >=80% | >=85% | >=85% | >=80% |
[/TABLE]
Note: Gate thresholds are NOT fixed — they vary between 80% and 85% across periods. Both gates must be met simultaneously. B2C provides the achievement data for both gates.

[TABLE]
| Factor | Campaign Hygiene | Distributor Accelerate Profitable |

8. Key Definitions & Glossary
[TABLE]
| Term | Definition |
| DD | Distribution House / Distributor — the recipient of incentive in all Distribution Accelerate SRFs. |
| GA | Gross Add — successful new SIM activation. Measured by FCD (First Call Date) in the promotion period. |
| FCD | First Call Date — the date on which a subscriber first makes a call after activation. Used as the activation gate. |
| RDB | Retail Database — the agent/retailer list snapshot. Month-end RDB is used for attribution in Campaign Hygiene. |
| SCR | Strike Rate — ratio of productive/active retailers at a Distribution House level. |
| DH SCR | Distribution House Strike Rate — SCR measured at the DH level. Shared as an eligibility gate. |
| LSO | Local Sales Officer — field force role at retailer-servicing level. Active LSO count is a KPI metric. |
| BSO | Business Support Officer — field force role. Active BSO count is a KPI metric. |
| SSO | Sales Support Officer — field force role added from May 2025 (replaced DSO in Apr 2025). |
| DSO | Distribution Support Officer — appeared in Apr 2025 SRF only; replaced by SSO from May 2025. |
| RSO | Retail Sales Officer — field force role. RSO wise daily minimum transaction count is tracked. |
| FF GA Participation | Field Force GA Participation rate — introduced in Aug 2025 Campaign Hygiene. |
| 48hr Usage | Usage within 48 hours of SIM activation — quality gate in Distributor Accelerate Profitable. Threshold: >=80% or >=85%. |
| VLR Active | Visitor Location Register Active — subscriber must be on VLR (i.e., active in the network) to count. Gate for Profitable programme. |
| TCG Bar | Technical/Commercial Grade Bar — barred connection type. Excluded from GA count in Dec 2025 onwards. |
| Self-Barred | Subscriber-initiated bar — excluded from GA count in Dec 2025 onwards. |
| ETSAF | End-to-End SAF compliance metric — 98% of activations must be vetted and accepted within 7th of following month (Campaign Hygiene). |
| EV OTF | Electronic Voucher Over The Form — a recharge method counted in the Recharge definition. |
| IRIS OTF | IRIS Over The Form — included in recharge definition for Campaign Hygiene. |
| DRC EV | Digital Recharge Center EV — specifically EXCLUDED from recharge definition in Campaign Hygiene. |
| R2RT | RSO/Retailer-to-Retailer Transfer — counted as EV sales in Campaign Hygiene recharge definition. |
| D2RT | Distributor-to-Retailer Transfer — counted as EV sales in Campaign Hygiene recharge definition. |
| C2S | Customer-to-Subscriber recharge — the valid recharge basis for Distributor Accelerate Profitable. |
| T-6 | Report generation date minus 6 days — used as the cutoff for disconnection data. |
| Daraz Voucher | E-commerce gift voucher (Daraz platform) — the incentive format for Distributor Accelerate Profitable. |
| Category A/B/C/D | Distributor performance tiers. Higher category = higher incentive value. Targets shared by B2C. |
| Slab 1/2/3 | Alternative tiering used in Jan 2026 Profitable phases. Values shared by B2C. |
| East Cluster Exception | In Jan 2026 Phase 1 only: East Cluster DDs eligible at >=70% achievement (vs >=100% for all other clusters/periods). |
| Max Consider | Rule in Jan 2026 Phase 2 and 3: DD is eligible for only 1 gift — the maximum applicable slab is awarded even if multiple slabs are met. |
[/TABLE]
