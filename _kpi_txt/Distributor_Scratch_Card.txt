1. Overview & SRF Coverage
[TABLE]
| Programme | Distributor Scratch Card Campaign |
| SRF Count | 10 SRFs (Dec 2025 – Apr 2026) |
| Recipient | DD — Direct Distributor |
| Product Type | Scratch Card (SC) |
| Campaign Types | Two types: (A) SC Lifting Discount — 7 SRFs; (B) SC Tertiary — 3 SRFs |
| Periodicity | Ad-hoc; each SRF covers a specific campaign window (multi-day, within a month) |
| Report Slot | Disbursement — end of campaign |
[/TABLE]
All campaigns are short-window, ad-hoc incentive schemes issued to distributors (DDs). Lifting campaigns reward DDs for purchasing (lifting) scratch cards from the warehouse. Tertiary campaigns reward DDs for ensuring scratch cards are recharged by end subscribers via BTS-wise SC recharge. Both types are independent and may run concurrently within the same calendar period.
2. Campaign Type Summary — Lifting vs Tertiary
[TABLE]
| Dimension | Type A: Lifting Discount | Type B: Tertiary |
| KPI Measured | Warehouse → Distributor SC lifting volume | DD BTS-wise SC recharge (Retailer → Subscriber) |
| Achievement Basis | Face value of SC lifted (cumulative) | Cumulative SC tertiary target (BTS-wise) |
| Incentive Basis | Face value or Lifting value (varies) | Amount shared by B2C (DD-wise target) |
| Commission Rate | 1% (most SRFs) or B2C-defined amount | B2C-defined amount (no fixed % in SRF) |
| Celebration Budget | None | TK 200 per RSO upon achieving target |
| Achievement Criteria | % achievement (threshold = 100%) | Hit/Miss (binary) |
| Cap | No cap / 150% (varies by SRF) | Not specified |
| SRF Count | 7 SRFs | 3 SRFs |
[/TABLE]
3. Campaign Type A: SC Lifting Discount Campaigns (7 SRFs)
Seven SRFs cover lifting discount campaigns across Dec 2025 – Mar 2026. While they share a common structure, there are meaningful variations in commission basis, cap rules, eligible card sets, and eligibility gate conditions.
3.1  Common Conditions — All Lifting Campaigns
[TABLE]
| Trigger | DD achieves 100% or above of SC Lifting target (Face value) cumulatively during campaign period |
| Lifting Direction | Warehouse to Distributor (W→D) only — not Distributor to Retailer |
| Measurement | Face value of selected SC products lifted during the campaign window |
| Rounding | General mathematical rounding: below 0.5 rounds down; 0.5 and above rounds up |
| Agent List Date | Last day of the campaign period (date stated in each SRF) |
| Commission | 1% on value (face or lifting) — amount shared by B2C for some SRFs |
[/TABLE]
3.2  Achievement vs Incentive Calculation Basis
Most lifting SRFs use Face Value for both achievement measurement and incentive calculation. However, one SRF (Jan26 1-12) uniquely uses Lifting Value for incentive calculation while retaining Face Value for achievement measurement:
[TABLE]
| SRF | Achievement Basis | Incentive Calculation Basis | Note |
| Dec25 (9-11 Dec) | Face value | Face value | Standard |
| Jan26 (1-12 Jan) | Face value | Lifting value | UNIQUE — incentive on lifting price, not face value |
| Jan26 (6-11 Jan) | Face value | Face value | Excludes SCV-39-60M-48H from incentive (see 3.7) |
| Jan26 (14-22 Jan) | Face value | Face value | Standard |
| Feb26 (1-8 Feb) | Face value | Face value | Standard |
| Feb26 (17-22 Feb) | Face value | B2C-defined | No % rate stated; eligibility gate applies |
| Mar26 (10-15 Mar) | Face value | Face value | 1% rate; broadest product list |
[/TABLE]
i  Lifting value = the actual price paid by the distributor (typically 96.25% of face value). Face value = the printed denomination on the scratch card.

4. Campaign Type B: SC Tertiary Campaigns (3 SRFs)
Three SRFs cover SC Tertiary campaigns (Feb 2026, Mar 2026, Apr 2026). All three are structurally identical — conditions are consistent across the full set.
4.1  Common Conditions — All Tertiary Campaigns
[TABLE]
| SRFs | Feb26 (17-26 Feb), Mar26 (7-12 Mar), Apr26 (16-22 Apr) |
| Trigger | Cumulative 100% and above SC Tertiary target achieved |
| Achievement Criteria | Hit/Miss — binary; no partial payment |
| Incentive Amount | DD-wise target and incentive amount shared by B2C (amount not stated in SRF) |
| Celebration Budget | TK 200 per RSO upon achieving SC tertiary target (amount shared by B2C) |
| Rounding | General mathematical rounding (below 0.5 down, 0.5+ up) |
| Agent List Date | Last day of campaign period |
[/TABLE]
4.2  Tertiary Metric Definition
[TABLE]
| Metric Name | SC Tertiary |
| Definition | DD designated BTS-wise SC recharge — recharge of scratch cards by end subscribers within the DD's designated BTS (Base Transceiver Station) coverage area |
| Data Source | Achievement data shared by B2C team |
| Direction | Retailer → Subscriber (end-user recharge of SC) |
| Difference from Lifting | Lifting = Warehouse → Distributor. Tertiary = Retailer → Subscriber. These are opposite ends of the supply chain. |
[/TABLE]
4.3  Celebration Budget Rule
[TABLE]
| Condition | DD achieves 100%+ of SC tertiary target (same trigger as main incentive) |
| Amount | TK 200 per RSO assigned to that DD |
| Source | Incentive amount shared/defined by B2C team |
| Unique to | Tertiary campaigns only — no celebration budget in any Lifting campaign |
| Payment flow | Included in disbursement; reported in same SRF |
[/TABLE]

9. Glossary
[TABLE]
| Term | Meaning |
| SC | Scratch Card — physical/digital prepaid voucher |
| DD | Direct Distributor |
| W→D Lifting | Warehouse to Distributor — primary purchase of SC stock by DD from Banglalink warehouse |
| SC Tertiary | Scratch card recharge by end subscriber via BTS-designated retailer (Retailer → Subscriber) |
| Face Value | Printed denomination of the scratch card (e.g., TK 29, TK 39) |
| Lifting Value | Actual price paid by distributor — typically 96.25% of face value (e.g., SCV-39-60M-48H at 37.5375) |
| BTS | Base Transceiver Station — geographic coverage unit for tertiary target allocation |
| Hit/Miss | Binary achievement criteria — 100%+ = Hit (paid); below 100% = Miss (no payment) |
| Cap 150% | Incentive ceiling — calculated only on up to 150% of target even if achievement is higher |
| Uncapped | No ceiling on incentive — full 1% on all face value regardless of achievement level |
| B2C | Business-to-Consumer team at Banglalink — defines target and incentive amounts for some campaigns |
| RSO | Retail Sales Officer — field agent; used for celebration budget calculation in tertiary campaigns |
| Celebration Budget | TK 200 per RSO paid in addition to main incentive upon achieving SC tertiary target |
| T-3 | Report generation date minus 3 days — disconnection cutoff for these ad-hoc SC campaigns |
| SCV | Scratch Card Voice product type |
| SCD | Scratch Card Data product type |
| SCMB | Scratch Card Mixed Bundle product type |
| MV | Mixed Voucher product type |
| DMS | Distribution Management System — Banglalink retailer/distributor registry |
[/TABLE]
