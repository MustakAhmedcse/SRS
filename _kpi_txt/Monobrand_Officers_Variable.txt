1. Overview & Epoch Timeline
This document extracts all unique KPI conditions from 11 SRFs in the Monobrand Officers Variable folder. The primary commission scheme (Variable Incentive) evolved through six distinct structural epochs between January 2025 and January 2026, with an additional standalone RYZE slab commission and a separate Reactivation Campaign.
Epoch Timeline
[TABLE]
| Epoch | Period | Key Changes |
| 1 | Jan–Mar 2025 | Ryze Migration (10%) + GA (10%) + iTOP (40%) + First Visit Resolution (20%). 100% SAF. |
| 2 | Apr 2025 | Ryze Migration KPI dropped. GA weight raised to 20%. 100% SAF. |
| 3 | May 2025 | Recharge & Bundle Upsell (20%), Daily Participation (20%), App Install (20%) added. SAF lowered to 98%. iTOP dropped. |
| 4 | Jul 2025 | Bundle ratio reduced to 10% (from 15%). Security deposits updated. |
| 5 | Oct–Dec 2025 | CSAT (15%) added. GA weight raised to 30%. Multi-slab payout (60%/70%/80%/90%/100%). First Recharge sub-KPI (rolling 45 days). Maturity: 45 days. |
| 6 | Jan 2026 | FWA lead generation (10%) added. GA threshold raised to 93%. New payout tiers (95%/110%/120%). CSAT split: Platinum+Signature (15%) + Overall (15%) = 30%. |
[/TABLE]
2. Fixed Incentive Structure (BDT 4,000 / 6,000)
Throughout all epochs, the fixed monthly variable incentive pool remains unchanged. KPI achievement determines what percentage of this pool each CCR receives.
[TABLE]
| Recipient Type | Fixed Variable Incentive (Performance-Based) |
| All Monobrand CCR | BDT 4,000 per month |
| Monobrand Store In-charge | BDT 6,000 per month |
[/TABLE]
Note: Actual payout = Fixed incentive × weighted KPI achievement score. Consistent across all 6 epochs.
3. Dual SAF Framework – Manual & ETSAF
Monobrand channel uniquely operates two SAF submission pathways simultaneously. Both must be tracked separately and have distinct deadlines and rules.
Manual SAF (Alternate ID / Alternative ID)
[TABLE]
| Parameter | Detail |
| Trigger | Alternate ID-based acquisitions from Monobrand Channel |
| Data Source | BSS Commission team considers submitted & Non-Submitted SAF from system |
| Deadline | Must be Cleared (vetted & accepted by CO Team) by T+6 (7th day from activation) |
| MNP | Not considered for Manual SAF commission |
[/TABLE]
ET SAF (ETSAF) Process
[TABLE]
| Parameter | Detail |
| Correction Window | ETSAF correction must be within 72 hours to be eligible |
| SAF Threshold – Epoch 1–2 (Jan–Mar 2025) | 100% SAF submission required. If count is 90, all 90 must qualify |
| SAF Threshold – Epoch 3–6 (May 2025+) | 98% SAF submission required. If count is 100, 98 must qualify |
| Zero Incentive Rule | If SAF qualifies below threshold, GA KPI = zero incentive for that CCR |
| MNP | Not considered in ETSAF commission |
[/TABLE]
Key SAF Change: Epoch 1–2 → Epoch 3+
[TABLE]
| Period | SAF Threshold | Effect |
| Jan 2025 – Apr 2025 (Epochs 1–2) | 100% SAF required | Any single unqualified SAF zeroes GA KPI |
| May 2025 onwards (Epochs 3–6) | 98% SAF required | Up to 2% SAF shortfall allowed before GA KPI zeroed |
[/TABLE]
4. Postpaid GA Eligibility & Security Deposit Rules
Postpaid connections are counted as GA only when all of the following conditions are met: activation and bundle activated on the same day, and the required minimum security deposit is paid in full by same day (can be multiple shots but same day). Only MNP and new acquisition are eligible — not renewal. The deposit thresholds evolved across epochs.

Jan26 Revision – CSAT Correction
A separate revision SRF was submitted (Apr 2026) covering only the Platinum & Signature CSAT sub-component (15%) for January 2026. The payout structure for this component is: Score <98% = Zero; Score 98–99% = 80%; Score >99–99.8% = 100%; Score >99.8% = 120%. Minimum 20 CSAT feedbacks/month required. This SRF is a correction only — no other KPIs changed.
